# caesarBrute

**Caesar Cipher Brute Forcer**

## How To Install ?

```
git clone https://github.com/brokenheartz/caesarBrute.git
cd caesarBrute
pip3 install -r requirementx.txt
chmod u+x caesar.py
./caesar.py
```

## How to use?

### Using cipher string
> **$ ./caesar.py \<cipher string\>**

__Example :__

> **$ ./caesar.py iyvrluolhyag**

### Using cipher file

> **$ ./caesar.py \<cipher file\>**

__Example :__

> **$ ./caesar.py cipher.txt**
